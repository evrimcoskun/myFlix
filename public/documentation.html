Movie API

a REST API for an application called “myFlix” that interacts with a database that stores data about different movies.

Methods

Method Path Description Body
GET /movies List of all movies -
GET /movies/[title] Get one move by title -
GET /users Get list of all users -
POST /users Create a new user Username: [Username],
Password: [Password],
Email: [Email],
Birthday: [Birthday]
PUT /users/[username] Update a user Username: [Username],
Password: [Password],
Email: [Email],
Birthday: [Birthday]
PUT /users/[username]/favorites/[id] Update one user's favorites -
DELETE /users/[username]/favorites/[id] Delete from one user's favorites -
DELETE /users/[username] Delete user -
